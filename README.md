# Installing

If you don't have rokit or npm / nodejs go to these links:
[rokit](https://github.com/rojo-rbx/rokit/releases/latest)
[nodejs](https://nodejs.org/en/download/prebuilt-installer)

You may have to restart after installing rokit.

Install rokit packages:
`rokit install`

Install nodemon:
`npm install nodemon`

# Building

Run `lune run scripts/start-dev` in your terminal to build and sync to rojo.

If you need debug profiling then change the `DEBUG_PROFILING` variable in `scripts/darkluaconfig` this is also where you will find all macros
